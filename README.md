# 🎯 内容选题助手

一个基于"选题四象限"方法论的内容选题生成工具，专为知识型IP（律师、心理咨询师、培训师、专家等）设计。

## 功能特点

- 📋 **配置一次，长期使用** - 首次设置后，信息保存在本地，无需重复输入
- 💡 **极简交互** - 只需输入灵感/想法，自动生成高质量选题
- 🎯 **精准定向** - 基于你的目标人群和产品，生成真正能转化的选题
- 🌐 **纯静态网页** - 无需服务器，可部署到任何静态托管平台
- 🔑 **无需配置API** - 智谱AI API已内置，开箱即用

## 使用方法

### 本地使用

直接用浏览器打开 `index.html` 文件即可使用。

### 在线部署（推荐）

#### GitHub Pages（免费）

1. 创建一个 GitHub 仓库
2. 上传 `index.html` 文件
3. 在仓库设置中启用 GitHub Pages
4. 分享生成的链接给任何人使用

#### Vercel（免费）

1. 访问 https://vercel.com
2. 导入 GitHub 仓库或直接拖拽文件
3. 自动部署完成

#### Netlify Drop（最简单）

1. 访问 https://app.netlify.com/drop
2. 直接拖拽 `index.html` 文件
3. 几秒后获得可用链接

## 使用流程

### 首次使用：配置信息

1. 打开网页
2. 填写配置信息：
   - 你是谁？你的专业领域是什么？
   - 你的目标人群是谁？（详细描述）
   - 你提供什么产品/服务？
   - 当前阶段（起步/成长/成熟）
   - 选题目标（涨粉/信任/转化）

### 日常使用：生成选题

1. 输入你的灵感、想法、热点或任何内容素材
2. 点击"生成选题"
3. 查看为四个象限生成的选题建议

## 选题四象限方法论

| 象限 | 特点 | 适用场景 |
|------|------|----------|
| 泛人群+泛选题 | 流量型，破圈潜力大 | 账号冷启动、快速涨粉 |
| 泛人群+垂直选题 | 专业破圈，建立权威 | 展示专业能力 |
| 垂直人群+泛选题 | 情感共鸣，精准连接 | 建立信任、深度连接 |
| 垂直人群+垂直选题 | 精准转化，直接变现 | 产品推广、服务成交 |

## 数据隐私

- 所有数据存储在你的浏览器本地（localStorage）
- 不会上传到任何服务器（除了调用智谱AI API）

## 技术栈

- 纯 HTML + CSS + JavaScript
- 智谱AI API（GLM-4-Flash 模型）
- 无需后端服务器
